# Movie App

A small React example project for browsing and saving favorite movies.

## Features

- Browse popular movies
- Search for movies
- Save favorites

## Tech Stack

- React 19
- Vite
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file with:

```
VITE_API_KEY=your_api_key
VITE_BASE_URL=https://api.themoviedb.org/3
```
